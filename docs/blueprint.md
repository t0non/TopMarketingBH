# **App Name**: Top Marketing BH

## Core Features:

- Vitrine da Página Inicial: Exiba uma seleção de projetos de sites concluídos com sucesso, demonstrando as habilidades de desenvolvimento web da agência.
- Página de Destino de Tráfego Pago: Crie uma página de destino dedicada para capturar leads de campanhas de tráfego pago com calls to action claros.
- Depoimentos de Clientes: Apresente depoimentos de clientes para construir confiança e credibilidade, destacando o impacto positivo da agência nos negócios.
- Descrições de Serviços: Detalhe os serviços de desenvolvimento web e gerenciamento de tráfego pago da agência, explicando o processo e os resultados esperados.
- Formulário de Contato: Forneça um formulário de contato simples para que clientes em potencial entrem em contato facilmente e solicitem informações sobre os serviços.
- Ferramenta de Geração de Conteúdo para Blog: Gere conteúdo de blog para clientes em potencial que demonstre a liderança de pensamento de marketing. Isso é alcançado através da entrada de palavras-chave pelo usuário relacionadas ao público-alvo. Esta ferramenta usará raciocínio para decidir se deve ou não incluir informações.

## Style Guidelines:

- Cor primária: Azul claro (#64B5F6) para transmitir profissionalismo e confiança, inspirado no branding estabelecido.
- Cor de fundo: Azul-acinzentado claro (#DCE6F2) para uma aparência limpa e moderna que complementa a cor primária.
- Cor de destaque: Vermelho mais claro (#EF5350) usado com moderação para principais calls to action, proporcionando uma sensação de urgência e energia.
- Fonte: 'Montserrat' (sans-serif) conforme solicitado pelo usuário.
- Nota: atualmente, apenas o Google Fonts é compatível.
- Use ícones modernos e minimalistas em vermelho para destacar os principais serviços e benefícios.
- Projete um layout limpo e estruturado que destaque tanto os projetos de desenvolvimento web quanto a expertise em tráfego pago, focando em uma experiência amigável.
- Incorpore animações sutis na página de destino e em todo o site para chamar a atenção para detalhes importantes e criar uma experiência dinâmica.
